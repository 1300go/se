## 📂 프로젝트 상세: 클라우드 파일 공유 시스템 (Mini Drive)

### 1. 주요 해결 과제
* **버전 관리**: 여러 사용자가 작업 시 최신 파일 식별의 어려움 해결
* **검색 효율성**: 분산된 파일 저장 경로로 인한 시간 낭비 방지
* **보안 강화**: 이메일/메신저 공유 시 발생하는 외부 유출 위험 차단

### 2. 핵심 기능 상세
* **중앙 관리형 스토리지**: 모든 업무용 파일을 한곳에서 안전하게 보관
* **스마트 폴더 구조**: 프로젝트별 다단계 폴더 구성을 통한 체계적 관리
* **정밀한 권한 제어**: 보기, 수정, 댓글 등 사용자별 차등 권한 부여
* **버전 히스토리**: 수정 이력 확인 및 필요시 이전 버전으로 즉시 복구
* **고성능 검색**: 파일명, 업로드 날짜, 유형별 필터를 통한 신속한 검색
